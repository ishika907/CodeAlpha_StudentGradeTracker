# Student Grade Tracker

## Description
A console-based Student Grade Tracker developed using Core Java. The application allows users to manage student records, calculate grades, and generate summary reports.

## Features
- Add Student
- Display All Students
- Delete Student
- Calculate Average Marks
- Find Highest Marks
- Find Lowest Marks
- Calculate Grade
- Menu-Driven Interface
- Exception Handling
- Input Validation

## Technologies Used
- Java
- Core Java
- OOP
- ArrayList
- Exception Handling

## Project Structure
```
StudentGradeTracker/
│
├── Main.java
├── Student.java
├── StudentService.java
└── README.md
```

## How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run `Main.java`.
4. Use the console menu to manage student records.
